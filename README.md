# Tuner

A simple violin tuner built with TanStack Start, React and TypeScript.

## Installation

1. Clone the repository:

```bash
git clone https://github.com/ct02k/tuner
cd tuner
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:3000` to use the tuner.
